# 🏨 Estudo de Caso: Sistema Pousadas Férias no Norte

## 📌 Sobre o Projeto
Este repositório apresenta a modelagem de sistemas para uma rede de pousadas fictícia. O objetivo é documentar a arquitetura lógica e os requisitos para um sistema de gestão de reservas, consumo e faturamento.

## 🛠️ Stack de Engenharia Definida
De acordo com as diretrizes do projeto, a implementação segue as seguintes tecnologias:
* **Linguagem:** Java (IDE Eclipse)
* **Persistência (ORM):** Hibernate
* **Banco de Dados:** MySQL
* **Padrão de Arquitetura:** MVC (Model-View-Controller)
* **Metodologia:** Scrum (Prazo: 4 meses)
